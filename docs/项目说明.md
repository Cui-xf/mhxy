# 项目说明

## 简介

本仓库是一个**以实践为主**的学习项目，目标是熟悉 **J2ME 工程**的逆向分析流程：从资源与字节码入手，结合反编译、运行环境 Hook 与文档化记录，理解一款 J2ME 游戏的结构与行为。

## 主要资源与目录

| 路径 / 文件 | 说明 |
|-------------|------|
| `java版梦回西游3区251011.jar` | 待分析的目标 **J2ME 游戏** JAR 包。 |
| `agent/` | **Java Agent** 工程，在运行时对游戏 JAR 进行 **Hook**（如网络收发、日志、方法插桩等），便于动态观察行为。 |
| `docs/` | 项目文档与分析笔记（含逆向结论、流程说明等）。 |
| `jadx/` | 使用 [JADX](https://github.com/skylot/jadx) 等工具从 JAR **反编译导出**的源码视图，便于静态阅读（与原始混淆/结构可能不完全一致，需对照字节码与运行验证）。 |
| `jdk8u472-b08/` | 用于**启动游戏**的 JDK 8 环境（**当前仅提供 Windows 版本**）。 |
| `KEmulatorLite/` | 基于 JDK 的 **J2ME 运行基座**，在 Windows 上作为模拟器运行 MIDlet / 游戏。 |
| `unzip/` | 将游戏 JAR **直接解压**后的目录（`META-INF`、`.class`、资源文件等），便于快速浏览结构与资源，不等同于完整“工程源码”。 |
| `note/` | 探索过程中的**随笔/备忘**，非正式文档，可按个人习惯记录尝试与结论。 |

## 使用思路（概要）

1. **静态**：结合 `unzip/` 与 `jadx/` 阅读类名、资源与调用关系。  
2. **动态**：在 `jdk8u472-b08` + `KEmulatorLite` 上运行目标 JAR，按需挂载 `agent/` 打好的 Agent 进行 Hook。  
3. **沉淀**：将可复用的结论写入 `docs/`，临时想法放在 `note/`。

具体启动参数、Agent 打包方式与 Hook 配置以 `agent` 工程与 `docs` 内专题文档为准。
